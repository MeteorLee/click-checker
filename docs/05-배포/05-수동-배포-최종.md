# 수동 배포 가이드

## 1. 목적

- 수동 배포를 1회 완주 가능한 절차로 고정한다.
- 자동 배포 설계 이전에 배포 흐름을 직접 이해한다.
- 실패 시 중단/복구 기준을 명확히 한다.

---

## 2. 사전 체크

- 배포 대상 브랜치 또는 커밋 확인
- 환경 변수(.env) 존재 확인
- 직전 정상 버전 확인 (롤백 대비)
- jq 설치 여부 확인

```bash
command -v jq >/dev/null || { echo "jq not installed"; exit 1; }
```

---

## 3. 배포 순서 (강제)

1. build
2. test
3. deploy
4. health 확인
5. 기능 스모크 확인

> 원칙: 앞 단계가 실패하면 다음 단계로 진행하지 않는다.

---

## 4. 실행 절차

### 4.1 Build

```bash
./gradlew clean bootJar --no-daemon
```

---

### 4.2 Test

```bash
./gradlew test --no-daemon
```

---

### 4.3 Deploy

```bash
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build app postgres prometheus grafana
```

---

### 4.4 Health 확인

```bash
curl -f http://localhost:8080/actuator/health
```

성공 기준:

- HTTP 200
- status = UP

---

## 5. 기능 스모크 (최소 실행 기준)

### 5.1 Organization 생성

```bash
ORG_JSON=$(curl -fsS -X POST http://localhost:8080/api/organizations \
  -H 'Content-Type: application/json' \
  -d '{"name":"deploy-check-org"}')

ORG_ID=$(echo "${ORG_JSON}" | jq -r '.id')

test -n "${ORG_ID}" && test "${ORG_ID}" != "null"

echo "ORG_ID=${ORG_ID}"
```

---

### 5.2 이벤트 저장

```bash
curl -fsS -X POST http://localhost:8080/api/events \
  -H 'Content-Type: application/json' \
  -d '{
    "organizationId": '"${ORG_ID}"',
    "eventType": "click",
    "path": "/__test/deploy-check",
    "occurredAt": "2026-02-28T00:00:00"
  }'
```

---

### 5.3 집계 조회

```bash
curl -fsS "http://localhost:8080/api/events/aggregates/paths?organizationId=${ORG_ID}&from=2020-01-01T00:00:00&to=2030-01-01T00:00:00&top=5"
```

성공 기준:

- 저장 요청이 2xx 응답
- 집계 조회가 200 응답

---

## 6. 실패 시 중단 / 복구 규칙

- build 실패 → 즉시 중단
- test 실패 → 즉시 중단
- health 실패 → 롤백
- 스모크 실패 → 롤백

---

## 7. 최소 롤백 절차

1. 직전 정상 커밋 확인

```bash
git log --oneline -n 5
```

2. 이전 버전으로 복귀

```bash
git checkout <LAST_GOOD_COMMIT>
```

3. 재배포

```bash
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build app postgres prometheus grafana
```

4. health 재확인

---

## 8. 완료 기준

- build → test → deploy → health 순서 위반 없이 완료
- 기능 스모크 통과
- 롤백 가능 상태 확보
